==============================
Gamepad Active
==============================

Detects if the controller at given index has any activity.

Inputs
++++++++++++++++++++++++++++++

Index
   The controller index to monitor.

Outputs
++++++++++++++++++++++++++++++

Active
   *True* if the controller has activity, else *False*.
