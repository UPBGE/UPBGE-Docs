=================
Cursor Visibility
=================

*Cursor Visibility* node is used to set the visibility status of the system mouse cursor.

Inputs
++++++

Condition
   Input condition.

Visible
   Target state for the system cursor.

Outputs
+++++++

Done
   *True* if the node performed successfully, else *False*.
