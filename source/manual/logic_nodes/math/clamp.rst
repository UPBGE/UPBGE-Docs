==============================
Clamp
==============================

Will clamp *Value* between *Min* and *Max* values.

Inputs
++++++++++++++++++++++++++++++

Value
   Value to clamp.

Min
   Minimum value to clamp to.

Max
   Maximum value to clamp to.

Outputs
++++++++++++++++++++++++++++++

Value
   Resulting clamped value.
