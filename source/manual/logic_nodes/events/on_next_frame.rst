=============
On Next Frame
=============

Delays a condition by one tick/frame.

Inputs
++++++

Condition
   If *True*, the condition is reserved until the next tick/frame.

Outputs
+++++++

Out
  *True* one tick/frame after the *Condition* socket is activated.
