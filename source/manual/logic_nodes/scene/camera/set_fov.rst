==============
Set FOV
==============

Will set a *Field of View* to selected camera.

Inputs
++++++

Condition
   If connected, condition must be fulfilled for node to activate.

Camera
   Which camera to set a FOV to.

FOV
   Value of FOV to set.

Outputs
+++++++

Done 
   *True* if the node performed successfully, else *False*.
