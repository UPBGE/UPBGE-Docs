
+++++++++++++++
Moved
+++++++++++++++

.. figure:: /images/Logic_Nodes/mouse_moved_node.png
   :align: right
   :width: 215
   :alt: Mouse Moved Node.

The Mouse -> *Moved* node detects mouse movement.

Parameters
==========

Mode
   Toggle between *Once* and *Each Frame* modes.

Outputs
=======

If Moved
   *True* if mouse movement is detected, else *False*
