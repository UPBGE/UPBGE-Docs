+++++++++++++++
Key Code
+++++++++++++++

.. figure:: /images/Logic_Nodes/key_code_node.png
   :align: right
   :width: 215
   :alt: Key Code Node.

The *Key Code* node retrieves a specified key's numeric code. This node is meant for
debugging purposes.

Inputs
=======

Key
   The key to translate.

Outputs
=======

Code
   Integer code of the selected key.
